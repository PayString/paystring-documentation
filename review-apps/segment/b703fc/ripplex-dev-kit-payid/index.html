Use RippleX Dev Kit With PayID

RippleX Dev Kit can be used to simplify the process of developing with PayID.

RippleX Dev Kit provides libraries in JavaScript, Java, and Swift.

Documentation#

RippleX Dev Kit provides classes and utilites to work with PayID. Head over to the PayID documentation for the JavaScript, Java, or Swift libraries to get started.

Demos#

RippleX Dev Kit provides a number of demos in their demo repo. You can browse PayID demos in JavaScript, Java, or Swift. For a quick run through the demo, see Get Started With RippleX Dev Kit and PayID.
