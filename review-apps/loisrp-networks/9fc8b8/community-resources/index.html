Community Resources

Members of the PayID community have created useful resources for using and learning about PayID.

Tutorials

  • PayID Youtube Tutorial by Ken Carabello

Validators

  • PayID Validator - An open-source project that validates responses using JSON schemas and verifies whether your PayID server follows various best practices, like:

    • HTTP response headers
    • Cache-Control
    • CORS accessibility
    • Response time

  • Python-based PayID validator - Validates URI strings to determine if they are valid PayIDs.

PayID server implementations

  • Django PayID Server - A Django-based PayID server.
  • PayID Static - A Node.js PayID server with static configuration (easy to use on Heroku).
