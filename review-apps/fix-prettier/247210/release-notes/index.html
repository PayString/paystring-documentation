Release Notes

note

PayString was previously known as PayID.

See release notes for the last releases:

  • PayString 1.4 - August 25, 2020 - Improvement of PayString CLI. Updates to payloads for the PayString Admin API for greater consistency.
  • PayString 1.3 - August 18, 2020 - Support for self-sovereign verifiable PayString.
  • PayString 1.2 - August 4, 2020 - Support for PayString Discovery, and the addition of the PATCH method to the Public API.

See all release notes on Github.
